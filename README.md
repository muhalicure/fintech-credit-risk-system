# Fintech Credit Risk System

A Python Tkinter application for credit risk assessment.

## Features

- Customer credit score calculation
- Credit application evaluation
- CSV data processing
- Simple graphical user interface with Tkinter
- Risk classification system

## Technologies Used

- Python
- Tkinter
- Pandas
- CSV

## Project Structure

- kredi_skorlama.py → Main application
- kredi_basvurulari.csv → Sample application data
